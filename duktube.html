Intro
- 유튜브 클론 코딩 프로젝트이다.
- Frontend는 react, apollo client를 사용하였고 backend는 apollo server, prisma를 사용하였다.
Login
- 로그인은 json web token을 사용하였다.
- 올바른 email, password를 입력하면 backend에서 json web token을 생성 후 front로 토큰을 전달한다.
- 전달받은 토큰은 localStorage에 저장을 하고 반응형 변수(makeVar)를 이용하여 만든 isLoggedInVar 변수를 true로 바꾸어 주었다.
Video detail
- 동영상 상세 페이지에는 동영상, 제목, 설명, 조회수 , 업로드한 사용자, 구독자 수, 댓글, 구독 또는 수정 버튼 등을 보여준다.
- 좋아요 버튼의 경우 누르면 backend에서 받은 isLiked(로그인한 유저가 누른적이있는지 true/false로 알려준다.)를 통해 누른적이 없는 경우 파란색으로 변한다.
- 좋아요 버튼을 누르면 옆의 좋아요 수도 useMutation의 update를 통하여 바꾸어 주기 때문에 새로고침 없이 바뀌는 것이 바로 보인다.
- 구독 또는 수정 버튼의 경우는 동영상의 주인이 자신이면 수정 버튼이 자신이 아닌 경우에는 구독 버튼이 보인다.
- 구독 버튼의 경우 좋아요 버튼과 마찬가지로 구독 또는 구독중으로 버튼이 바뀐다.
- 동영상의 조회수는 동영상 길이의 90%의 시간이 지나면 올라가도록 하였다.
- 동영상의 댓글은 작성 후 refetchQueries를 통해 작성한 댓글도 다시 받아와 출력해 주었다.
User detail
- 사용자의 프로필 사진 또는 사용자 명을 누르면 사용자의 상세 페이지를 보여준다.
- 사용자의 구독자 수와 사용자의 동영상을 보여주고 구독 버튼을 보여준다.
