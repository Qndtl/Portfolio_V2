Simple Posting
Intro
간단한 텍스트 게시물을 올리고 좋아요, 댓글 기능이 있는 프로젝트이다.
이 프로젝트는 MongoDB, Express, React, GraphQL, Apollo Server를 사용하였다.
Login은 passport, jsonwebtoken을 사용하였다.
User
Sign up은 email과 username을 입력하고나면 MongoDB에 사용자가 등록이 된다.
등록된 이메일로 로그인을 시도하면 User에 loginSecret을 생성하고 등록된 이메일에 mailgun을 사용하여 loginSecret을 보낸다.
loginSecret을 복사하여 로그인이 완료되면 localstorage에 json web token이 저장되어 로그인을 유지한다.
Upload
Add Post 버튼을 누르면 작성할 수 있도록 modal 창이 뜬다.
@apollo/client의 useMutation을 사용하여 Post를 등록한다.
등록한 Post에 Like, Comment, Update, Delete 버튼이 존재하는데 로그인한 자신의 것에만 Update, Delete버튼이 존재한다.
Comment & Like
말풍선 버튼을 누르면 Comment들과 새로운 Comment를 작성할 수 있도록 modal 창이 뜬다.
Comment는 useMutation을 사용하여 등록한다.
Like는 Post의 id와 Like를 누른 User의 id가 존재하는 경우 Like를 삭제하고, 없는 경우에는 생성하여 Like 여부를 useQuery로 불러와 Like 상태를 보여준다.
