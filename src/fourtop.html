Intro
3D 프린터 커뮤니티 및 렌탈 서비스를 하는 프로젝트이다.
Frontend는 React를 사용하였고 Backend는 Prisma2를 사용하였다.
Login
로그인은 json web token을 사용하여 localStorage에 토큰을 저장하는 방식을 사용하였다.
로그인은 아이디와 비밀번호를 이용하고 bcrypt를 이용하여 비밀번호를 저장하였다.
로그인에 성공을 하면 jwt 토큰을 보내주어 localStorage에 저장을 하고 redux의 state도 변경해준다.
Redux는 아직 배우고 있어 redux-persist를 이용하여 로그인 상태를 유지해 주었다.
Gallery
이달의 무료 작품은 좋아요 많은 순 5명만 보여준다.
이달의 유료 작품은 준비중입니다.
이달의 크리에이터는 좋아요, 댓글 많은 순 5명만 보여준다.
Company
useState를 사용하여 소개, 인사말, 비젼 & 미션, 연혁 탭을 클릭하면 각각의 컴포넌트를 보여준다.
Upload
사진은 AWS S3에 저장을 하고 Prisma의 Post에는 AWS S3에 저장되어있는 사진의 url이 저장되어있다 (multer, multer-s3사용)
이미지 파일을 선택하면 선택한 이미지 파일을 미리보여준다.
Profile
사용자의 게시물 수, 팔로워 수, 팔로잉 수 등을 볼 수 있다.
프로필이 자신인 경우에는 edit profile이 보여 프로필 변경이 가능하다.
팔로워와 팔로우를 누르면 팔로워와 팔로우하는 사용자들을 오른쪽에 보이도록 하였다.
